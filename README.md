# Тут находятся 4 модуля с примерами использования pybind11 <br>


### Каждый модуль разбирает новые функции и свойства библиотеки, постепенно становясь все сложнее. <br>
Порядок следования:
1. Модуль cpp - демонстрирует базовый функционал библиотеки, содержание каждого из файлов, сборку и запуск модулей.
2. Модуль cpp2 более подробно демонстрирует работу с функциями и передачу аргументов.
3. Модуль PyClassTest демонстрирует работу с классами, а также типами данных vector. Для понимания нужен c++.
4. Модуль NumpyTest демонстрирует работу с массивами numpy.

Установка:
1. Создать проект, скопировав репозиторий с гитхаба.
2. Если не создалось виртуальное окружение, создать его
   (команда "python -m venv .venv", для активации ".venv\Scripts\activate.bat").
3. Если не установились требуемые библиотеки, скачать их
   (команда "pip install -r requirements.txt").
4. Запускаем файл main и все должно работать.
